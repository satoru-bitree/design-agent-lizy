Key things to remember about Lizy:
- **Korean-first** product. English reserved for the wordmark and system tags.
- **Studio-dark, single-accent** mood — near-black canvas, mint `#00C896` is the only chromatic accent, white text.
- **No emoji, no gradients in chrome, no decorative illustrations.** Elevation comes from surface lightness, not shadow.
- **Brand lockup**: italic Fraunces "Lizy" (mint `y`) + green-gradient rounded-square mark chip. Subtitle "Design Agent" in small-caps Korean-stack font with wide tracking.
- **Type**: Manrope (display) + Inter (body) + Pretendard (Korean) + Fraunces italic (wordmark only) + JetBrains Mono (numerals/hex).
- **Iconography**: Lucide line icons, 1.5px stroke, rounded joins. Substitution flagged in README.
- **Voice**: concise, declarative, sensory; polite imperative (-하세요/-합니다); status-first labels.
